# Python Learning Repository

A well-structured Python repository demonstrating fundamental to intermediate skills through beginner-friendly examples. It showcases coding basics, logical thinking, and practical problem-solving.

## Repository Structure
- Basics/: Core Python fundamentals
  - Variables and data types
  - Conditional statements
  - Loops
  - Basic input/output
- Intermediate/: Builds on the basics
  - Functions
  - Lists and dictionaries
  - String manipulation
  - File handling
- Projects/: Small practical projects applying learned concepts

## Purpose
- Demonstrate foundational and intermediate Python skills
- Show clean code organization
- Highlight problem-solving and practical application

## How to Run
Each Python file can be executed individually:
`bash
python filename.py
