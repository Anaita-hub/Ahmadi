
# Student Grade Analyzer 📊

This is a simple advanced Python project using pandas for data analysis.

## Features
- Load student grades from CSV
- Calculate statistics (average, max, min)
- Count passed/failed students
- Add letter grades (A-F)
- Export processed data

## Requirements
pip install pandas

## How to Run
python main.py

## Output
- Displays summary in terminal
- Saves processed file as output.csv
